=============================================
How to use:

Modify FUNCTION_LIST file (by un-comment of your function name) 
+ ARM:
run  "PFC_setting_generator_arm.csh FUNCTION_LIST <file_name>"
+ Example:
PFC_setting_generator_arm.csh FUNCTION_LIST msi0_arm

Note:
- If you not specify <file_name>, result will be
   + PFC_setting_arm.src (for ARM).
